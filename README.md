# Habit Tracker App – UI/UX Design (Task 2)

## Project Overview
This project is a UI/UX design for a Habit Tracker Mobile Application.
The app helps users build positive daily habits, track progress, and stay motivated through a simple and user-friendly interface.

This project was designed as part of an internship-style UI/UX design task using Figma.

## App Name
MindFit – Habit Tracker App

## Objective
- To help users build and maintain daily habits
- To allow users to track habit completion easily
- To motivate users through progress and streak tracking
- To design a clean and minimal habit tracking interface

## User Flow
Welcome Page  
↓  
Home Page  
↓  
Dashboard / Tracker Page  
↓  
Activity Details Page  
↓  
Profile Page  

## Pages Designed

### Welcome Page
- Displays app logo and welcome message
- Introduces the purpose of the app
- Guides the user to start using the app

### Home Page
- Welcomes the user with a greeting
- Displays today’s habit progress
- Shows motivation quote
- Includes button to view today’s habits

### Dashboard / Tracker Page
- Displays list of daily habits
- Each habit includes a checkbox or toggle
- Allows users to mark habits as completed
- Includes option to add a new habit

### Activity Details Page
- Shows selected habit name and description
- Displays progress details such as completed days and streak
- Includes action buttons like:
  - Mark as Completed
  - Undo Completion
  - Set Reminder

### Profile Page
- Displays user information
- Shows overall habit statistics
- Allows users to manage account and preferences
